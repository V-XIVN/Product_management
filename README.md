# Product_management
 产品进存销管理系统

## 文件目录
#### 头文件
1. [account.h](/source/account.h)
    * 提供账户类的原型以及方法定义
2. [Store.h](/source/Store.h)
    * 提供仓库类的原型
#### 源文件
3. [Store.cpp](/source/Store.cpp)
    * 提供仓库类方法定义

* 产品操作

* 产品变动记录
    * 添加
